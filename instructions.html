Getting started

Instructions

A quick tour of the core concepts, then a guided first workflow.

1. Create a workspace

Sign in and click New workspace. Workspaces are isolated environments — credentials, workflows, and runs don't leak between them.

2. Connect a tool

Open Integrations, choose a provider, and authorize. Whirl stores credentials encrypted; you can revoke at any time.

3. Build your first workflow

Drag a trigger onto the canvas, add actions, connect them, hit Test. When the run looks right, click Publish.

4. Watch it run

Every execution shows up in Runs with full logs, timing, and retry controls.
