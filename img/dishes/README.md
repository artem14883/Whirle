# Фото страв Суші Шінобі

Сюди закидайте справжні фото своїх страв. Сайт автоматично підставить кожне фото за іменем файлу. Якщо файлу немає — використається стокове фото з Unsplash як підстава.

## Формат

- **Розмір:** не менше 480×480 px (можна більше, бажано квадрат)
- **Формат:** `.jpg` або `.png`
- **Імена:** усі **малими латинськими літерами** + дефіси, **без пробілів і кирилиці**

## Як це працює

1. Кладете фото у цю папку з правильним іменем
2. Робите `git add img/dishes/нова-страва.jpg`
3. Робите `git commit -m "Add photo"`
4. Робите `git push`
5. За ~1 хвилину сайт оновиться, ваше фото з'явиться в картці

Будь-яке фото без імені у цьому списку буде проігноровано — використається стокове.
