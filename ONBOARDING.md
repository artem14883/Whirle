# Суші Шінобі — сайт кафе

Односторінковий сайт для суші-кафе «Шінобі» (Корсунь-Шевченківський, Україна) з 3D-анімаціями на Three.js.

## Що в репозиторії

- [`index.html`](index.html) — головна сторінка (Hero, Меню, Сет тижня, Про нас, Відгуки, Контакти, Футер)
- [`css/sushi.css`](css/sushi.css) — стилі (темна тема, японські акценти, повністю адаптивна верстка)
- [`js/sushi.js`](js/sushi.js) — Three.js 3D-сцени + інтерактив:
  - фонова сцена: частинки + torus knot з паралаксом
  - hero-сцена: 6 ролів по колу + центральний рол з гарніром
  - DOM-пелюстки сакури, scroll reveal, card tilt
  - mobile burger nav
- старі файли з шаблону Whirl (`blog.html`, `pricing.html`, тощо) — **можна видалити**, до нового сайту відношення не мають

## Як запустити

```bash
# через будь-який статичний сервер
npx serve -l 3000 .
# або
python -m http.server 3000
```

Відкрити: <http://localhost:3000>

У Claude Code є preconfigured launcher — команда `preview_start whirle` піднімає сервер автоматично.

## Адаптивність — перевірені брейкпоінти

- **≥ 1024px** — повна сітка: 3 колонки меню, 2 колонки в hero/sets/about/contact
- **768px (планшет)** — 2 колонки меню, hero/sets/about/contact в одну колонку
- **≤ 560px (мобільний)** — burger-меню, всі секції в одну колонку, частинки і сакура зменшені

## Типові задачі

- **Додати позицію в меню** → секція `.menu__grid` в `index.html`, додати `<article class="dish">` за зразком (категорія, назва, опис, ціна, вага)
- **Змінити Сет тижня** → секція `#sets`, оновити заголовок, склад, ціни
- **Поміняти 3D-роли** → масив `fillingPalette` у `js/sushi.js` (heroScene)
- **Поміняти швидкість анімацій** → константи в `tick()`-функціях у `sushi.js`

## Стек

- Чистий HTML / CSS / JavaScript — без збірки, без npm-залежностей
- Three.js r128 з CDN (cdnjs.cloudflare.com)
- Шрифти: Noto Serif JP (японські акценти) + Manrope (основний текст), Google Fonts
- Google Maps embed для карти
